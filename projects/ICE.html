ICE-BUTTON (SaaS)

  • Tech Stack: Twilio, Raspberry Pi, Django, RTMP Protocol, Nginx

Developed a secure IoT video streaming system using Raspberry Pi, enabling RTMP-based video transmission from IoT devices.

Integrated Nginx and RTMP protocols to allow seamless browser-based video streaming while ensuring a secure and scalable infrastructure.

Designed and implemented robust Django-based APIs for managing admin functionalities, payment gateways, communication features, and multi-tenancy support for the SaaS platform.

Ensured efficient backend scaling by optimizing load balancing and API performance, improving system reliability and reducing latency.
